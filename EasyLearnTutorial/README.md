Easy Learn Tutorial
===================

A native Android version of my tutorial website http://www.easylearntutorial.com


USE CASES
---------

[Visitor] - views and selects
  * List of tutorial series
  * List of tutorials in series
  * Individual tutorial

[Visitor] - shares
  * Individual tutorial series
  * Inidvidual tutorial


VIEWS
-----

  * List of tutorial series
  * List of tutorials
  * Tutorial


ENTITIES
--------

[Tutorial]
  * id
  * title
  * thumbnail
  * description
  * video_id
  * date

[Series]
  * id
  * title
  * thumbnail
  * description
  * tutorials
